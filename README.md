# 赫子 ✕ 中登 — 流体显形 hero

一个 WebGL2 流体显形单页：静止显示「赫子」，鼠标搅动流体处把它冲刷开，露出底下的「中登」。红黑白（FCC）配色，零依赖，单文件。

- 移动鼠标：流体涂抹 + 原位换脸（赫子 ↔ 中登）
- 鼠标纵向牵引：上移露下半身 / 下移人像整体下沉
- 液态流动感来自流体模拟，脸部保持清晰

## 结构
- `index.html` — 全部逻辑（着色器 + 流体模拟 + 交互），引用两张图
- `hezi.jpg` / `zhongdeng.jpg` — 两张对齐的半身像（同构图，一张真脸一张特效脸）

顶部 `TUNE` 可调：`portraitH`（大小）、`portraitOffY`（高低）、`panYGain`（牵引幅度）、`smear`（液态扭曲）、`zdShiftY`（中登对齐）。

## 本地
用任意静态服务器打开（WebGL 在 `file://` 下会拦截外部图片，需走 http）：
```
python -m http.server 8250
```

原创 WebGL2 · 复用自个人站 lando-hero 的流体显形实验。
